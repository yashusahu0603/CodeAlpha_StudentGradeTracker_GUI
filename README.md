# CodeAlpha - Student Grade Tracker (GUI)

## Description
A Java Swing GUI application to manage student grades.

## Features
- Add Student
- Display Student Report
- Calculate Average Marks
- Find Highest Marks
- Find Lowest Marks
- Grade Calculation
- Pass/Fail Status

## Technologies Used
- Java
- Java Swing
- VS Code

## How to Run

Compile:

```bash
javac StudentGradeTrackerGUI.java
```

Run:

```bash
java StudentGradeTrackerGUI
```
